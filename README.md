# 🚀 Gemini Clone

A modern AI-powered chat application inspired by Google Gemini, built with a sleek UI and real-time conversational capabilities. This project replicates core Gemini-like features including prompt-based interactions, dynamic responses, and a clean responsive interface.


## ✨ Features

- 💬 Real-time AI chat interface

- 🎨 Clean and responsive UI

- ⚡ Fast and optimized performance

- 🔐 Secure API handling


## 🛠️ Tech Stack

**Frontend**
- React.js
- Tailwind CSS

**API**
- Gemini API


## ⚙️ Installation

### 1. Clone the repository
```bash
git clone https://github.com/username/project.git
cd project
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the project
```bash
npm run dev
```
